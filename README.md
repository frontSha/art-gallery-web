# Art Gallery
Mini proyecto de galería de arte que permite buscar y explorar obras de arte usando la API del Instituto de Arte de Chicago.

**Demo:** https://frontsha.github.io/art-gallery-web/

## Objetivos
- Practicar el consumo de APIs
- Organizar código usando JavaScript modules

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript ES6+ (modules *import/export*, fetch API, manejo básico de estado)

## Funcionalidades
- Búsqueda de obras
- Render dinámico de resultados
- Navegación por páginas
